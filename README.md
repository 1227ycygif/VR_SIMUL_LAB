# VR_SIMUL_LAB
VR 화학 실험실 안전 교육 시뮬레이션 

## 프로젝트 소개 
가상현실 환경에서 화학 실험실의 안전 수칙을 체험하고 학습하는 교육용 시뮬레이션  

## 담당 역할 (Technical Lead) 
- 파티클 시스템 구현 
- 사운드 시스템 설계
- 환경 구성 및 최적화

## 기술 스택
- Unity 
- C# 
- VR(Meta Quest) 
- Particle System 
- Audio System 

## 주요 기능
- VR 환경 내 실험 기구 조작
- 파티클 효과 (화학 반응, 위험 요소 시각화)
- 공간 오디오 시스템
- 안전 교육 수칙 

## 개발 기간 
2025.11 ~ 2025.11 (2주/게임부트캠프)
